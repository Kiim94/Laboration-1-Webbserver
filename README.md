Länk till webbplats: https://laboration-1-webbserver.onrender.com/

Diagram över tabellen finns ovan, en png fil. Inte diagram.drawio - den råkade komma med också.

Det kan ta en liten stund för webbplatsen att "starta" - har använt Render.

Uppgiften har gått ut på att skapa ett databas-baserat CV med Express, EJS och PostgreSQL (den senare var vad jag valde att använda). 
En startsida med tillagda kurser, en sida med formulär som man kan fylla i information i samt en about sida där jag delat lite tankar om det hela. 

Det svåraste med uppgiften har varit att lära sig allt nytt - det är inte särskilt likt JavaScript med POST, GET, DELETE med mera.
